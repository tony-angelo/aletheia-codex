================================================================================
DNS QUICK REFERENCE FOR aletheiacodex.app
================================================================================

STEP 1: UPDATE NAMESERVERS AT YOUR REGISTRAR
--------------------------------------------------------------------------------
Replace your current nameservers with these Google Cloud DNS nameservers:

    ns-cloud-c1.googledomains.com
    ns-cloud-c2.googledomains.com
    ns-cloud-c3.googledomains.com
    ns-cloud-c4.googledomains.com

STEP 2: WAIT FOR DNS PROPAGATION (1-48 hours, typically 2-4 hours)
--------------------------------------------------------------------------------
Check propagation status:
    dig aletheiacodex.app NS
    
Or use online tools:
    https://www.whatsmydns.net/
    https://dnschecker.org/

STEP 3: VERIFY DOMAIN OWNERSHIP
--------------------------------------------------------------------------------
Option A: Google Search Console
    1. Go to: https://search.google.com/search-console
    2. Add property: aletheiacodex.app
    3. Choose DNS verification
    4. Add TXT record provided by Google
    5. Click Verify

Option B: Firebase Console
    1. Go to: https://console.firebase.google.com/project/aletheia-codex-prod/hosting/sites
    2. Click "Add custom domain"
    3. Enter: aletheiacodex.app
    4. Follow verification steps

STEP 4: ADD DNS RECORDS
--------------------------------------------------------------------------------
After Firebase provides IP addresses, add these records:

Main Site (A Records):
    Type: A
    Name: @
    Value: [IP addresses from Firebase]
    TTL: 3600

    Type: A
    Name: www
    Value: [IP addresses from Firebase]
    TTL: 3600

API Subdomain (CNAME):
    Type: CNAME
    Name: api
    Value: ghs.googlehosted.com.
    TTL: 3600

Graph API Subdomain (CNAME - Optional):
    Type: CNAME
    Name: graph
    Value: ghs.googlehosted.com.
    TTL: 3600

STEP 5: MAP DOMAINS TO CLOUD RUN
--------------------------------------------------------------------------------
After domain verification:

    gcloud beta run domain-mappings create \
      --service=review-api \
      --domain=api.aletheiacodex.app \
      --region=us-central1

    gcloud beta run domain-mappings create \
      --service=graph \
      --domain=graph.aletheiacodex.app \
      --region=us-central1

STEP 6: TEST
--------------------------------------------------------------------------------
Main site:     https://aletheiacodex.app
API endpoint:  https://api.aletheiacodex.app/health
Graph API:     https://graph.aletheiacodex.app/health

AUTOMATION SCRIPT
--------------------------------------------------------------------------------
Run this script to automate DNS configuration:
    cd aletheia-codex
    ./DOMAIN_SETUP_AUTOMATION.sh

TROUBLESHOOTING
--------------------------------------------------------------------------------
Check DNS records:
    dig aletheiacodex.app A
    dig api.aletheiacodex.app CNAME
    dig aletheiacodex.app NS

Check domain verification:
    gcloud domains list-user-verified

Check domain mappings:
    gcloud beta run domain-mappings list --region=us-central1

TIMELINE
--------------------------------------------------------------------------------
Nameserver update:     5 minutes
DNS propagation:       1-48 hours (typically 2-4 hours)
Domain verification:   5-10 minutes
DNS records setup:     10-15 minutes
Domain mapping:        5-10 minutes
SSL certificate:       1-48 hours (automatic)
Total:                 2-96 hours (typically 4-8 hours)
